breaking up alarm clock into componenets
setting an alarm (time, ringer (song, beat))
snooze the alarm (reset ot 5 later, ignore altogether)


Start
1. Grab the alarm clock
2. Create the alarm
    a. Set the time for the alarm (AM/PM)
    b. Set the desired ringer (song, beat)
3. Set the alarm (make sure that it if confirmed and set for correct day and time)

when alarm rings at set time:
1. Click snooze
    a. Give 5 more minutes
    b. Give 10 more minutes

    repeat until alarm stopped:
        once extra time is up:
            a. Ask again for x more minutes
            b. Stop the alarm

2. Stop the alarm
